## 📂 Projects

### 🎯 Project 1 — Number Guessing Game

A console-based Java application where the user attempts to guess a randomly generated number.

#### Features

* Random number generation
* User input handling
* High/Low feedback system
* Continuous gameplay loop
* Input validation and exception handling

#### Concepts Used

* Random Class
* Loops
* Conditional Statements
* Exception Handling
* User Interaction

---

### 📊 Project 2 — Student Grade Calculator

A Java application that calculates total marks, average percentage, and final grade based on student performance.

#### Features

* Multiple subject input
* Total marks calculation
* Percentage calculation
* Automatic grade assignment
* Clear result display

#### Concepts Used

* Variables
* Arithmetic Operations
* Loops
* Conditional Logic
* Data Validation

---

### 🏧 Project 3 — ATM Interface

An Object-Oriented Java application that simulates ATM operations using separate classes for account management and transaction handling.

#### Features

* Deposit money
* Withdraw money
* Balance inquiry
* Input validation
* Insufficient balance checks
* Modular OOP design

#### Concepts Used

* Classes & Objects
* Encapsulation
* Methods
* Object-Oriented Programming (OOP)
* Input Validation
